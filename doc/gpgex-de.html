GnuPG Shell Extension

(Verfügbare Sprachen: English, Deutsch)

Die GNU Privacy Guard Erweiterungen für die Windows Explorer Umgebung erlaubt es ihnen, kryptographische Funktionen in Kleopatra über die Kontext-Menüs von Dateien aufzurufen.

Befehle

Entschlüsseln und verifizieren, Entschlüsseln, Verifizieren

Entschlüsselt und/oder verifiziert die ausgewählten Dateien. FIXME: Hier wird ein Link zur Kleopatra-Dokumentation für die Entschlüssel- und Verifizier-Funktionen eingestellt.

Verschlüsseln und signieren, Verschlüsseln, Signieren

Verschlüsselt und/oder signiert die ausgewählten Dateien. FIXME: Hier wird ein Link zur Kleopatra-Dokumentation für die Verschlüssel- und Signier-Funktionen eingestellt.

Zertifikate importieren

Import die Zertifikate in den ausgewählten Dateien. FIXME: Hier wird ein Link zur Kleopatra-Dokumentation für die Import-Funktion eingestellt.

Prüfsummen erstellen

Erstellt Prüfsummen für die ausgewählten Dateien. FIXME: Hier wird ein Link zur Kleopatra-Dokumentation für die Funktionen zur Erstellung von Prüfsummen eingestellt.

Verifiziert die Prüfsummen für die ausgewählten Dateien. FIXME: Hier wird ein Link zur Kleopatra-Dokumentation für die Funktion zur Verifikation von Prüfsummen eingestellt.

Debugging

Falls nicht auf Kleopatra zugegriffen werden kann, wird ein Nachrichtenfenster mit einer Fehlermeldung angezeigt. Um mehr Informationen zu erhalten, kann der Fehlerbericht von Kleopatra eingesehen werden, oder es kann eine Debug Datei im Registry Eintrag GpgEX Debug File zum Schlüssel HKLM\Software\GNU\GnuPG angegeben werden. Informationen zum Protokoll werden dann ggf. nach einem Neustart an diese Datei angehangen.
